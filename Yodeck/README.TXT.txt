1. Install JDK
Download and install the JDK:
Visit Oracle JDK or OpenJDK.
Select the version suitable for your operating system and install it.

#################################Set Up Required Plugins###############################################
2. Set Up IntelliJ IDEA
Open IntelliJ IDEA.
Install the Cucumber Plugin:
Navigate to File > Settings > Plugins.
In the Marketplace tab, search for "Cucumber for Java".
In the Marketplace tab, search for "Cucumber +".
Click Install and restart IntelliJ IDEA if prompted.

#################################Run tests ###############################################

3. Run the Tests
Open the project in IntelliJ IDEA.
Ensure all dependencies are added in the pom.xml file if the project uses Maven.
Locate the TestRunner.java file:
There is a folder in the following path: src/test/java/testRunner.
This file contains the configuration for running tests (e.g., feature file, reports).
Right-click on the TestRunner.java file and select Run 'TestRunner.

4. Check the Results
Test results will appear in the IntelliJ IDEA Run tab.
Find and click the test-output folder and check the results in the PDF and Yodeck_Daily_analysis.

#################################Have Fun###############################################
